Class Management

Next-generation discipleship tracking. Built for speed, secured for excellence.

The 1–3 Step Rule

Every primary task must be completed in 3 or fewer user-visible actions. Complexity is handled invisibly at the "Session Start" to empower our servants with lightning-fast scanning.

Core Actions
Level 1: Enrollment

Enroll Students

Goal: One-time "tagging" of a student to a class level.

01
Secure Access

Single gate (Auth + PIN) opens the portal.

02
Select Level

Choose Life Class, SOL 1, 2, or 3.

03
Scan

Scanner activates immediately and tags students.

Invisible System Logic

  • PIN requested once per session duration
  • Automated camera activation on level selection
  • Instant database synchronization
  • Haptic feedback on successful tagging
Level 2: Attendance

Take Attendance

Goal: Rapid Sunday morning check-in at scale.

01
Start Session

Enter PIN and confirm auto-filled details.

02
Continuous Scan

Scan everyone (Students and Staff) in one flow.

Smart Logic Engine

  • Pre-fills current week based on date
  • Defaults to 'Normal Mode' automatically
  • Auto-resolves Role (Student vs Staff) per scan
  • Cross-references enrollment tags in real-time
Control: Staffing

Staff Assignment

Goal: Direct admin control over roles and teams.

01
Select User

Unified search in Admin Dashboard.

02
Assign & Save

Set Role and Class Level instantly.

Back-office Logic

  • Bypasses scanner/PIN requirements
  • Audit log of all staff changes
  • Immediate propagation to scanner nodes
